README:
Para la correcta ejecución del proyecto siga las siguientes instrucciones:
1) Copie la carpeta “proyectoCrisly” en el disco local C, la carpeta debe encontrarse la raíz C para funcionar.
2) Ejecute el archivo de Python “ProyectoLenguajes.py” en su IDE favorito, recomendado PyCharm.
3) Una vez ejecutado el programa de Python diríjase a la carpeta “proyectoCrisly” recién ubicado en C y dentro de ella ubique el archivo “index.html”
4) Abra este archivo utilizando Google Chrome.
5) El botón “Solución” iniciará la animación.
